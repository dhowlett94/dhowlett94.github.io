What I've done

Unity GameJam Game

For my final year Team Project module I've been working alongside 2 other programmers and several designers. As part of the module, we took part in a month long game jam where we were tasked to create any game we liked, but it had to be in some way related to Mars. The idea we came up with was the Elon Musk Terraforming Game! With a game style similar to Cookie Clicker, you have to terraform mars by clicking on the planet in order to earn money and buy more gadgets to help you terraform!

Current Projects

For my Final Year Project, I am working to convert British Sign Language to speech. To do this, I am using a Leap Motion controller which performs the detection and tracking of the hands and then exposes that data via its API. This API data will be used with machine learning algorithms in order to accurately recognise the different signs. The application itself will be written in C# and to implement these machine learning algorithms, I will be using EmguCV (a C# wrapper around the OpenCV library). I also intend to use a SQL Server database in order to store the data for the different signs. For the speech component of the system, I will be using a free text-to-speech library, most likely the Speech Synthesizer already built into the .NET framework.
